MARCO-BOLO CSV-to-JSON-LD Tool

This repository contains a metadata publishing tool developed for Work Package 1 (WP1) of the MARCO-BOLO project (MARine COastal BiOdiversity Long-term Observations). WP1 focuses on data literacy and metadata flow across the project.

Learn more about WP1 here: https://marcobolo-project.eu/work-packages/work-package-1

This tool helps researchers and data managers transform metadata from structured CSV files into JSON-LD conforming to Schema.org and ready for harvesting by the ODIS Catalog.

Project Context

WP1 of MARCO-BOLO supports the production of high-quality, FAIR metadata. This repository provides the reference toolchain for transforming project metadata into a format compatible with global discovery platforms such as ODIS.

How It Works

The tool uses:

  • LinkML to define metadata models
  • CSV-W metadata files to describe CSV structure
  • Makefile and GitHub Actions to automate validation and transformation
  • W3IDs for stable context and schema URIs
  • Schema.org as the target vocabulary

Workflow Summary

  1. Fill in CSV templates (e.g. Dataset.csv, Person.csv)
  2. Commit changes; this kicks off GitHub Actions
  3. Download the JSON-LD files as an artifact
  4. Host the JSON-LD at a public URL
  5. Register it with ODIS

Getting Started (No Installation Required)

  1. Create a GitHub Account at https://github.com/signup. Contact WP1 to join the organization.
  2. Visit GitHub.dev Editor and install the Excel Viewer extension.
  3. Edit CSV files directly in-browser.
  4. Commit & push changes to trigger validation.
  5. Download JSON-LD output from the Actions panel.
